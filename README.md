一頁式網站，使用vue3撰寫
供買家有一個瀏覽商品環境

## 專案技術

- Node.js v14.16.0
- Vue v3.2.13
- pinia v2.1.7
- Tailwind v3.4.3
- element-plus v2.7.0
- gsap v3.12.5
- swiper v6.8.4
- aos v2.3.4
- vue-awesome-swiper 5.0.1
- vue3-table-lite 1.4.0

- modules - 模組放置處
- assets - 靜態資源(圖片)放置處
- store - pinia放置處(未使用)
